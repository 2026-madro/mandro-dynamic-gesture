# mandro 앱 용어 정의

> 비개발자 및 비전문가 사용자도 이해할 수 있도록  
> 기술 용어를 앱 내 표시 언어로 통일한 가이드  
> 작성일: 2026-06-29

---

## 1. 하드웨어 용어

| 기술 용어 | 앱 표시 용어 | 설명 (앱 내 안내 문구용) |
|-----------|-------------|--------------------------|
| Teensy (마이크로컨트롤러) | **암밴드** | 팔에 착용하는 기기 |
| EMG Sensor | **EMG 센서** / **센서** | 근육 신호를 읽는 장치 (그대로 사용) |
| Firmware | **기기 업데이트** | 암밴드에 새 설정을 저장하는 과정 |
| USB Flash / Upload | **기기 업데이트** | "암밴드를 USB로 연결해 업데이트합니다" |
| BLE (Bluetooth Low Energy) | **블루투스 연결** | 암밴드와 휴대폰을 무선으로 연결 |
| 8-channel | 표시 안 함 (또는 **8개 센서**) | 일반 유저에게 채널 수는 노출 최소화 |

## 2. 동작 (Gesture) 용어

영어 명칭을 기본으로 사용하되, 각 동작마다 **한 줄 설명**을 병기한다.

| 동작 | 앱 표시 | 한 줄 설명 |
|------|---------|-----------|
| Rest | **Rest** | 손을 자연스럽게 펴고 편하게 쉬기 |
| Flexion | **Flexion** | 손목을 아래로 구부리기 |
| Extension | **Extension** | 손목을 위로 젖히기 |
| Close (Fist) | **Close** | 손가락을 모아 주먹 쥐기 |
| Pronation | **Pronation** | 손바닥이 아래를 향하도록 돌리기 |
| Supination | **Supination** | 손바닥이 위를 향하도록 돌리기 |

> 학습 가이드 화면(S06)에서는 동작명 아래에 한 줄 설명을 항상 함께 표시한다.

## 3. 학습 / 소프트웨어 용어

| 기술 용어 | 앱 표시 용어 | 비고 |
|-----------|-------------|------|
| Model | **나만의 설정** | "내 동작 패턴을 학습한 개인 설정" |
| Training / Learning | **학습** | "내 동작을 학습하고 있어요" |
| Data Collection | **동작 녹화** | "동작을 따라 하면 앱이 기록해요" |
| Scaler / Normalization | 표시 안 함 | 내부 처리, 유저에게 노출 안 함 |
| Feature Extraction | 표시 안 함 | 내부 처리 |
| Inference / Classification | **동작 인식** | "암밴드가 동작을 인식하고 있어요" |
| Confidence Score | **정확도** | "인식 정확도 94%" |
| Session | **학습 기록** | "이전 학습 기록이 있어요" |
| Re-training | **다시 학습하기** | "동작 인식이 잘 안 되면 다시 학습해보세요" |

## 5. 앱 내 문구 톤 가이드

사용자 폭이 넓기 때문에 (절단 장애인, 재활 환자, 연구 참여자 등) 아래 원칙을 따른다.

### 원칙
- **쉬운 말** — 의학 용어, 개발 용어 최대한 배제
- **친근한 톤** — 딱딱하지 않게, 존댓말 유지
- **구체적인 안내** — "잠시 기다려 주세요" 보다 "암밴드에 설정을 저장하고 있어요 (약 30초)"
- **불안 줄이기** — 오래 걸리거나 낯선 과정엔 이유 설명 추가

### 좋은 예 / 나쁜 예

| ❌ 피할 표현 | ✅ 권장 표현 |
|------------|------------|
| 펌웨어를 업로드합니다 | 암밴드에 내 설정을 저장하고 있어요 |
| 모델 학습 중 | 내 동작 패턴을 학습하고 있어요 |
| BLE 연결 실패 | 암밴드를 찾지 못했어요. 가까이 두고 다시 시도해보세요 |
| Training complete: 94.2% | 학습 완료! 동작 인식 정확도 94% |
| Feature extraction | (표시 안 함) |
| Teensy not detected | USB 케이블이 연결되지 않았어요 |
| 데이터 수집 | 동작 녹화 |
| 서버에 전송 중 | 설정을 만들고 있어요 |
| 서버 업로드 완료 | 거의 다 됐어요! |
| 학습 완료: 94.2% | 완성됐어요! 인식 정확도 94% |
| Take 4/10 | 4번째 녹화 중 (총 10번) |
| Scaler not found | (내부 오류, 유저에게 "다시 학습해보세요"로 안내) |

## 6. 화면별 주요 문구 정리

### S01 · 시작 화면
```
앱 이름: mandro
부제: 내 동작을 기억하는 암밴드
```

### S02 · 홈
```
제목: 안녕하세요 👋
부제: 유저를 선택하거나 새로 만들어 주세요
연결 안내: 암밴드가 연결되지 않았어요
```

### S03 · 유저 생성
```
제목: 처음 오셨나요?
부제: 이름을 입력하면 나만의 설정을 만들어드려요
필수 동의: 내 동작 데이터로 개인 설정을 만들어요
선택 동의: 더 나은 인식을 위한 연구에 참여할게요 (선택)
```

### S04 · 블루투스 연결
```
제목: 암밴드 연결
부제: 근처의 암밴드를 찾고 있어요
발견: 아래 기기를 선택해 연결하세요
실패: 암밴드를 찾지 못했어요. 전원이 켜져 있는지 확인해보세요
```

### S06 · 동작 가이드
```
제목: [동작명] (예: Flexion)
부제: [한 줄 설명] (예: 손목을 아래로 구부리기)
안내: 아래 동작을 따라 해보세요
주의: 팔꿈치는 고정하고 손목만 움직여 주세요
완료: 이해했어요!
```

### S07 · 동작 녹화
```
제목: 동작 녹화
진행: [동작명]을 따라 해주세요
카운트다운: 3, 2, 1 → 지금!
유지: 3초간 유지해 주세요
완료: 잘 하셨어요 ✓
재수집: 다시 녹화하기

채널 끊김 경고 (발생 시에만 표시):
"암밴드 연결이 불안정해요. 다시 착용해보시겠어요?"
```

### S07-B · 녹화 품질 리포트 (신규)
```
제목: 녹화를 확인했어요
부제: 일부 동작은 다시 녹화하는 게 좋을 것 같아요

동작별 상태:
  ✓ Rest          정상
  ✓ Flexion       정상
  ⚠ Extension     "암밴드 연결이 불안정했어요"  [다시 녹화하기]
  ✓ Close         정상
  ✓ Supination    정상
  ✓ Pronation     정상

모두 정상일 때: "학습 시작하기" 버튼 바로 활성화
문제 있을 때: 해당 동작 재녹화 후 다시 확인
```

### S08 · 학습 진행
```
제목: 내 설정을 만들고 있어요
단계:
  ① 녹화 데이터 확인 중
  ② 설정을 만들고 있어요
  ③ 내 동작 패턴을 분석하고 있어요
  ④ 거의 다 됐어요!
완료: 완성됐어요! 인식 정확도 [N]%
```

> 서버 전송, 업로드, 네트워크 등 기술 용어는 이 화면에서 일절 노출하지 않는다.
> 유저 입장에서 내부 처리 방식은 중요하지 않으므로 "만들고 있어요 → 분석하고 있어요 → 거의 다 됐어요"
> 흐름으로 자연스럽게 이어지게 한다.

### S09 · 기기 업데이트
```
제목: 암밴드 업데이트
부제: USB 케이블로 암밴드를 연결해 주세요
체크: USB 케이블 연결됨 ✓ / 암밴드 감지됨 ✓ / 내 설정 준비됨 ✓
진행: 암밴드에 내 설정을 저장하고 있어요... (약 30초)
완료: 업데이트 완료! 이제 암밴드를 사용할 수 있어요 🎉
```

### S10 · 동작 인식
```
제목: 동작 인식 중
현재 동작: [동작명]
정확도: [N]%
낮을 때: 인식이 잘 안 되나요? → 다시 학습하기
```

## 7. 오류 메시지 가이드

오류 메시지는 **무슨 일이 생겼는지 + 어떻게 하면 되는지** 두 가지를 항상 함께 알려준다.

| 상황 | 표시 문구 |
|------|----------|
| BLE 연결 실패 | 암밴드를 찾지 못했어요. 암밴드 전원을 확인하고 다시 시도해 주세요. |
| BLE 연결 끊김 | 암밴드와 연결이 끊어졌어요. 가까이 두고 다시 연결해 주세요. |
| USB 미연결 | USB 케이블이 연결되지 않았어요. 케이블을 연결한 후 다시 시도해 주세요. |
| 업데이트 실패 | 업데이트 중 문제가 생겼어요. 케이블을 다시 연결하고 시도해 주세요. |
| 서버 연결 실패 | 인터넷 연결을 확인해 주세요. 연결 후 다시 시도해 주세요. |
| 학습 실패 | 설정을 만드는 중 문제가 생겼어요. 다시 시도해 주세요. |
| 녹화 데이터 부족 | 녹화 데이터가 부족해요. 각 동작을 10번씩 녹화해 주세요. |
| 인식 정확도 낮음 | 동작 인식이 잘 안 되고 있어요. 다시 학습하면 더 나아질 수 있어요. |
